What is Git and Github?

Git is something that is stored in the local computer. Github is something that is stored on cloud.

For example Photo Gallery on phone is Git and Google Photos app is Github.

Photo Gallery is saved on local storage inside the phone and cannot be retirved in case you lose your phone whereas Google photos app is always saved on the cloud and hence it is more reliable because we can access the same on any device and hence Git & Github works like this.
